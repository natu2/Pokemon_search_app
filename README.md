This is a Pokemon search app which allows a user to find information on a particular Pokemon by entering its ID or name. Information displayed includes its name, ID, height, weight, types and base stats. 

This app utilizes JavaScript to fetch data from various external api endpoints provided by FreeCodeCamp and HTML+CSS to display and render the fetched data.

## Install & Run
1. Clone this repository to your local machine
2. Open 'index.html' in your browser
3. Use the search bar to find the pokemon of your choice using either their name or Id.
